/* Title: Install Boost di Mac (Snow Leopard) Date: 2012/07/13 Page: post Labels: c++, boost */

Boost adalah standard library C++. Digunakan untuk membuat aplikasi cross-platform. Sebelum instalasi Boost pastikan XCode Developer Tool sudah terinstalasi. XCode diperlukan untuk meng-kompilasi source code "MacPorts" -- MacPorts, aplikasi untuk memudahkan instalasi pada mac, seperti apt-get pada ubuntu. Instalasi XCode juga memuat compiler dan library standard C++.

Download MacPorts dan ikuti cara instalasinya disini. Setelah selesai kita bisa melanjutkan instalasi Boost. Disini tidak dibahas cara instalasi MacPorts.

Ok, Ready. Buka terminal dan install Boost dengan MacPorts,

$ sudo port install boost

Tunggu prosesnya, lumayan lama. File yang didownload ukurannya diatas 90Mb.

Sudah selesai? Jika sudah kita lanjutkan. Library Boost secara otomatis telah terinstal difolder /opt/local/include/boost/.Cara menggunakannya, pertama buat link "boostlib" yang mengarah ke folder /opt/local/include/. Misal, folder penyimpanan kode program ada di /home/nama_user_mu/cpp_source_code/

$ cd /home/nama_user_mu/cpp_source_code
$ ln -s /opt/local/include/ boostlib

Setelah perintah diatas berhasil, link boostlib akan tersedia didalam folder. Link boostlib ini yang akan digunakan sebagai opsi include folder oleh g++.

Untuk mulai menggunakan boost, simpan kode C++ dibawah ini dengan nama first_boost.cpp:

#include <boost/lambda/lambda.hpp>
#include <iostream>
#include <iterator>
#include <algorithm>

int main()
{
  using namespace boost::lambda;
  typedef std::istream_iterator in;

  std::for_each(
    in(std::cin), in(), std::cout << (_1 * 3) << " " );
}

Kemudian diterminal jalankan perintah

$ g++ -I boostlib first_boost.cpp

Jika kompilasi sukses, maka intalasi Boost pun sukses. Jalankan aplikasinya, a.out adalah file executable yang dihasilkan kompiler g++

$ ./a.out

Ketik sembarang angka, dan aplikasi akan mengalikannya dengan angka 3.
