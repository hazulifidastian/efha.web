/* Title: Minimalis Date: 2016/03/18 Page: post Labels: profil */

Saya suka web design minimalis. Baik dari segi layout maupun komposisi warnanya. Minimalis juga berarti to the point (menurut saya). Mempermudah pengungjung untuk mendapatkan informasi, dan meminimalkan distraksi.

Web profil ini juga dibuat dengan konsep yang sama (minimalis). Contentnya hanya akan menampilkan yang penting-penting saja. Blog juga dibuat sesederhana mungkin.

Untuk memenuhi target, saya berburu nyari inspirasi diberbagai tempat. Maklum, karena keahlian dasar ada dibackend developer. Untuk urusan design kurang terlalu mantap.

Beberapa waktu berburu, minggu dan berbulan-bulan. Dapat inspirasi dari beberapa tempat.

Typeplate, Framework yang dipakai untuk mengorganisasikan struktur dan design sebuah web. Framework ini menekankan sisi typhography.

thepurdman.com, Web pribadi Kevin Purdman, penulis lepas. Designnya minimalis full text.

Monospace Writer, Aplikasi minimalis diandroid untuk membuat catatan / tulisan. Salah satu ide yang saya copy, kemampuan merubah themes antara terang dan gelap.

Web profil ini sepenuhnya dibangun dengan aplikasi dan framework open source, source codenya pun saya tempatkan di github. Silahkan jika ada yang berminat baca dan mengembangkan.

Untuk urusan backend engine nya, PICO file based CMS (content management system) jadi pilihan. Lagi-lagi dipilih karena simple. Tidak perlu setup database seperti MySql, content disimpan dalam bentuk file. Jika mau membuat post baru, cukup menambahkan file baru saja. Jika mau backup data, cukup zip satu folder saja.

Beberapa plugin sederhana saya buat untuk menambahkan fungsi yang dibutuhkan. Seperti label pada setiap post, script pembaca cokies untuk gonta ganti themes, dan plugin untuk menambahkan ".html" diakhir url.

Terakhir, thanks buat inspirator. Silahkan akses account github saya jika berminat melihat source codenya.
