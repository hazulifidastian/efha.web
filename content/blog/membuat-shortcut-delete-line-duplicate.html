/* Title: Membuat Shortcut Delete-Line dan Duplicate-Line di Xcode 4 Date: 2012/11/07 Page: post Labels: xcode */

Terbiasa menggunakan Netbeans untuk editing kode Php, saya begitu familiar dengan keyboard shortcutnya. Move line up/down & left/right, duplicate-line, dan delete-line adalah shortcut yang sangat populer. Di Xcode move line sudah tersedia, hanya perlu merubah key binding nya agar sesuai dengan default pada netbeans. Sedangkan untuk duplicate-line dan delete-line perlu sedikit usaha.

Buka terminal,

$ cp /Applications/Xcode.app/Contents/Frameworks/IDEKit.framework/Versions/A/Resources/IDETextKeyBindingSet.plist ~/Desktop/IDETextKeyBindingSet.plist

IDETextKeyBindingSet.plist adalah xml file yang menyimpan informasi shortcut yang digunakan oleh Xcode. IDETextKeyBindingSet.plist di salin ke desktop untuk di edit, karena file di direktori aslinya terkunci. Selanjutnya buka IDETextKeyBindingSet.plist yang ada di desktop dengan mengunakan texteditor, seperti textedit dan tambahkan baris:

# key dan dict ini tidak perlu ditambahkan, 
# hanya sekedar panduan untuk memasukkan key Customize

<key>Undo/Redo</key>
<dict>
  <key>Redo</key>
  <string>redo:</string>
  <key>Undo</key>
  <string>undo:</string>
</dict>

# masukkan key dan dict ini.
<key>Customized</key>
<dict>
  <key>Delete Line</key>
  <string>moveToEndOfLine:, deleteToBeginningOfLine:, deleteToEndOfParagraph:</string>
  <key>Duplicate Line</key>
  <string>selectLine:, copy:, moveToEndOfLine:, insertNewline:, paste:, deleteBackward:</string>
</dict>

Setelah disimpan, salin kembali file IDETextKeyBindingSet.plist yang sudah di edit ke direktori aslinya.

$ sudo cp ~/Desktop/IDETextKeyBindingSet.plist /Applications/Xcode.app/Contents/Frameworks/IDEKit.framework/Versions/A/Resources/IDETextKeyBindingSet.plist

Buka Xcode :Menu: Preferences :: Key Bindings. Gunakan fasilitas "Search" untuk mencari key yang baru ditambahkan (Delete Line dan Duplicate Line). Dan jika sudah ketemu, rubah kombinasi shortcut keyboardnya. Untuk shortcut yang conflict bisa cari kombinasi shortcut yang lain.

Diatas adalah cara manual menambahkan shortcut pada Xcode. Cara lain yang lebih mudah bisa dengan download aplikasi Keyboard Maestro.
