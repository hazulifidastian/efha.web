/* Title: Instalasi wxWidgets di Mac OS X dan XCode Date: 2012/11/05 Page: post Labels: c++, wxWidgets */

wxWidgets adalah library GUI C++ yang memudahkan developer membuat aplikasi dengan arsitektur 32-bit dan 64-bit pada platform Windows, OS X, Linux dan UNIX. Tidak seperti perangkat cross-platform lain, wxWidgets memberikan tampilan aplikasi benar-benar seperti tampilan aslinya (truly native look and feel). Karena wxWidget menggunakan API asli dari masing-masing platform tersebut.

Requirements

Instalasi wxWidgets di jalankan pada platform Mac OS X 10.8 dengan XCode 4.4. Default instalasi XCode 4.4 tidak menyertakan command-line tools untuk menjalankan perintah kompilasi seperti gcc dan g++. Paket command-line tools harus didownload secara terpisah dengan menjalankan XCode :Menu: Preferences :: Download :: Components :: Cari Command-line Tools dan selanjutnya klik "Check and install Now".

Kemudian dibutuhkan wxWidget. wxWidgets 2.9.4 yang dapat didownload di http://www.wxwidgets.org/downloads. Saat tulisan ini dibuat, versi terakhir wxWidget adalah 2.9.4. Pastikan yang didownload adalah link "With Unix line endings (for Unix and OS X): tar.bz2". Setelah download selesai, extract file ke direktori apa saja. Contoh, folder ~/wx/wxWidgets-2.9.4 .

Build the library

Ganti direktori ke WX root. Di contoh ini direktori WX root ada pada ~/wx/wxWidgets-2.9.4

$ cd ~/wx/wxWidgets-2.9.4
# perintah pada terminal Mac OS X

Create an output directory for the build, Setiap tipe build harus memiliki direktori terpisah dari source code.

$ mkdir build-release
$ cd build-release

Konfigurasi library

Sekarang konfigurasi library sesuai dengan spesifikasi tipe yang diinginkan. Untuk melihat opsi yang bisa digunakan pada Mac OS X, Possible Configure Flags under OS X . Untuk Mac OS X 10.8 opsi yang digunakan seperti ini:

$ ../configure --with-osx_cocoa --with-macosx-version-min=10.7 --with-macosx-sdk=/Applications/Xcode.app/Contents/Developer/Platforms/MacOSX.platform/Developer/SDKs/MacOSX10.7.sdk --prefix="$(pwd)"

Make

Kompilasi kode wxWidgets dengan perintah:

$ make
