/* Title: Export CSV Dengan Custom Delimeter Date: 2014/01/29 Page: post Labels: php */

Microsoft excel memiliki kemampuan untuk meng-expor data menjadi file csv. Standar csv hasil expor excel menggunakan delimeter atau pemisah koma (,). Namun untuk kebutuhan tertentu perlu menggunakan pemisah titik koma (;).

Tidak ada opsi khusus untuk merubah dari koma ke titik koma, kecuali dengan bersusah payah merubah regional setting dicontrol panel windows. Jika sekali atau dua kali tidak menyulitkan. Jika sering, bisa berganti menggunakan fungsi macro ini.

Option Explicit
    Sub save2csv()
    Dim lastColumn As Integer
    Dim lastRow As Integer
    Dim strString As String
    Dim i As Integer, j As Integer
    lastColumn = ActiveSheet.UsedRange.Column - 1 + ActiveSheet.UsedRange.Columns.Count
    lastRow = ActiveSheet.UsedRange.Rows(ActiveSheet.UsedRange.Rows.Count).Row
    
    Open "output.csv" For Output As #1
    
    For i = 1 To lastRow
    Cells(i, 1).Select
    strString = ""
    For j = 1 To lastColumn
    If j &lt;&gt; lastColumn Then
    strString = strString &amp; Cells(i, j).Value &amp; ";"
    Else
    strString = strString &amp; Cells(i, j).Value
    End If
    Next j
    Print #1, strString
    Next i
    
    Close #1
    
End Sub
